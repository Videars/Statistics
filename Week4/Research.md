# Research [^1]
> Formal properties of relative frequencies.

Relative frequency has some basic properties:

+ $0\leq f_A \leq 1$
+ $f_{A \cup B} = f_A + f_B $ in case $A$ and $B$ are disjoint
+ $f_{A \cup B} = f_A + f_B - f_{A \cap B}$
+ f( \0 ) = 0
+ f(P) = 1
